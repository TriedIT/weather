# Weather Information App By Piyush Mehta

## File structure

```structure
|-- weather-app
   |-- views
      |-- index.ejs
   |-- public
      |-- css
         |-- style.css
   |-- package.json
   |-- server.js
```
## How to deploy this app?

```bash
$ npm install npm@latest -g
$ node -v
$ git clone https://github.com/piyush97/weather.git
$ cd weather
$ node server.js
// Weather App is running on port 3000
```
